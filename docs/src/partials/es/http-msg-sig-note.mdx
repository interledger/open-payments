:::note[Firmas de mensajes HTTP]
Las Firmas de mensajes HTTP protegen la integridad de las solicitudes de API de Open Payments. El Kit de desarrollo de software (software development kit, SDK) TypeScript de Open Payments incluye la <LinkOut href='https://github.com/interledger/open-payments-node/tree/main/packages/http-signature-utils'>biblioteca Utils de la Firma HTTP</LinkOut>, que proporciona herramientas para trabajar con las Firmas de mensajes HTTP. Encontrará información adicional en las páginas de [Claves de cliente](/es/identity/client-keys) y [Firmas de mensajes HTTP](/es/identity/http-signatures).
:::
