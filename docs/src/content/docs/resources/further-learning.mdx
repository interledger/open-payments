## Test Network

Would you prefer to test Open Payments instead? The Test Network is a sandboxed environment that implements the Open Payments standard. It allows you to experience the entire payment cycle using fake money.

### The Test Network consists of:

- The <LinkOut href='https://wallet.interledger-test.dev/'>Test Wallet</LinkOut>: a digital wallet that implements the Open Payments APIs and allows you to send Interledger payments to any recipient with a payment pointer.{' '}

- The <LinkOut href='https://boutique.interledger-test.dev/products'>Test Boutique</LinkOut>: an e-commerce application that allows you to purchase virtual items using your Test Wallet.{' '}
